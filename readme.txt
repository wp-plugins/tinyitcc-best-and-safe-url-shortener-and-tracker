== Description ==

The wordpress plugin for the best and safe URL shortener and tracker - Tinyit.cc. Will automatically shorten the post's URL/Permalink and put the short link at the bottom of each post. Also supports posting your short link to Twitter. With Tinyit.cc, you convert long links to short tiny links, customize and share them instantly with your friends, get comprehensive click statistics for your short links and use free API in your applications. And the short URL's are safe. Each URL is checked before redirecting against Google safe browsing list of websites, phishtank phishing list, spamming list, SURBL list and Tinyit.cc blocked websites list. More information at http://tinyit.cc/more.html.
Dont have an API key? Register and get it now at http://tinyit.cc/getapi.html

== Installation ==

1. Upload the `Tinyit_shorturl` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You can change the default short URL service in 'Settings' -> 'Tinyit Short URL'
